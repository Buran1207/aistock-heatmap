# v8 评分透明化与二级交易评分重构说明

## 版本目标

v8 的目标是把系统从“显示评分结果”升级为“领导能看懂评分为什么这么评、因子如何量化、结果如何验证”。

核心改动：

1. 三个阶段页面内嵌评分方法与量化规则。
2. 解禁不再作为独立评分，只作为二级交易评分中的风险扣分和动作限制。
3. 二级交易评分改为：技术结构 + 量价确认 + IPO 锚点 + 相对强弱 + 流动性 - 风险扣分。
4. 技术指标先转成交易状态，再转成评分和买卖触发条件。
5. 行情新鲜度改为行情来源、最新交易日、更新状态和异常原因。
6. 估算解禁只作复核提示，不进入正式二级评分。
7. 评分体系与模型校准页集中展示方法论、权重方案、硬性风控上限和回测校准逻辑。

## 页面改动

### A1 项目观察池

新增“评分方法与量化规则”折叠区，展示：行业与港股偏好、公司稀缺性、同类 IPO 历史表现、中介与发行能力、未来交易可做性、市场窗口。

明确 A1 不使用发行价、认购倍数、孖展、暗盘、首日表现、上市后行情，避免未来函数。

### 招股期参与决策

新增“招股参与分如何计算”折叠区，展示发行定价、需求热度、资金效率、基石质量、发行结构、中介执行和市场窗口。

强调认购和孖展不是越高越好：极热 + 高定价 + 小流通盘可能是拥挤风险。

### 二级市场交易状态机

新增“二级交易评分、技术状态、买卖触发”折叠区，展示：

- 二级交易总公式
- 0-180D 和 180D+ 权重
- 趋势确认、放量突破、缩量回踩、深 V 确认、强势但过热、放量滞涨、破发弱势、高位回撤预警等技术状态定义
- 解禁如何作为二级风险扣分
- 硬性风控上限

### 二级风险：解禁供给解释

原“解禁与供给压力”页改为解释页，不再代表独立评分。正式二级评分只使用 iFind 精确解禁事件；估算事件只进入复核提示。

### 评分体系与模型校准

集中展示三阶段评分方法，并继续保留权重方案、评分分布和回测验证。

## 二级评分核心逻辑

二级交易评分 = 技术结构分 + 量价确认分 + IPO锚点分 + 相对强弱分 + 流动性分 - 风险扣分。

风险扣分包括：

- iFind 精确解禁压力
- 低流动性
- 行情未更新或非主源
- 过热
- 放量滞涨
- 高位回撤
- 数据置信度不足

高分不一定立刻买入。系统会区分：

- A 二级趋势确认：可参与，优先等回踩或成交确认
- B 二级交易观察：小仓或等待确认
- C 等待二级买点/风险释放：不追高
- D 破发/弱势回避：原则上回避
