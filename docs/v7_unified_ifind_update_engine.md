# v7 统一 iFind 日度更新引擎

## 核心原则

所有股票类 API 不再直接使用全港股主板或单个样例股票。

统一流程：

1. 用 `p05310 首发信息一览` 生成 2024 年以来已上市 IPO 股票池。
2. 清洗股票代码：过滤 H 临时代码、`_1/_2` 衍生代码、非四位正式港股代码。
3. 按批次取数：
   - `THS_HQ`：港股日行情。
   - `THS_RQ`：收盘快照。
   - `THS_BD`：行业/概念、财务摘要、股权结构、南向持股。
4. 本地缓存，失败时保留旧数据。
5. 输出到 `deploy_data/`，供 Streamlit 使用。

## 与 v6 的区别

- v6：日行情和快照命令里保留全港股主板长列表，脚本再筛选。
- v7：日行情和快照也在调用前替换为 2024+ IPO 股票池，减少额度消耗。

## 已接入的新增 API

- 港股指数行情：固定指数，不使用股票池。
- 行业分类 / 概念标签：`THS_BD`，动态股票池。
- 财务摘要：`THS_BD`，动态股票池。
- 股权结构 / 主要股东：`THS_BD`，动态股票池。
- 南向持股 / 港股通持股：`THS_BD`，动态股票池。

## 暂缺但已预留

- 估值指标。
- 公告目录 / 事件日历。
- 业绩日历。

## 解禁与技术分析

本版新增 `scripts/build_lockup_risk.py`，在 iFind 精确解禁明细基础上输出：

- `deploy_data/accurate_lockup_events.csv`
- `deploy_data/lockup_risk_model.csv`

风险评级不再一刀切，而是参考：

- 解禁市值 / 20 日成交额。
- 股价较发行价浮盈。
- 解禁类型。
- 距离解禁天数。
- 数据字段映射置信度。

专业技术分析继续由 `scripts/build_technical_signals.py` 输出，并合并进二级交易池。
