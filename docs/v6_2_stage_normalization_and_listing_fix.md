# v6.2 阶段口径与上市状态修复

本版修复内容：

1. 全站将“生命周期阶段”统一改为“阶段”。
2. 删除前台展示中的“半新股交易期”“次新延伸期”等说法，上市天数分层仅显示：
   - 0-30D
   - 31-180D
   - 180D+
3. “已上市”作为统一阶段口径，覆盖所有已经正式上市的 2024+ IPO 公司；不再用“已上市/半新股”作为阶段名称。
4. 前端增加上市日兜底校验：只要 listing_date <= 当前日期，即使旧 CSV 或 iFind 导出里仍写“招股/待上市”或“通过聆讯”，系统也会强制归类为“已上市”。
5. 构建脚本同步修复阶段口径，后续重新生成 deploy_data 时也不会再写入“已上市/半新股”。
6. 下一版继续处理：
   - iFind 字段映射正式接入主更新流程；
   - 精确解禁事件标准化与影响规律模型；
   - 专业技术评分纳入二级交易评分；
   - 评分科学性验证与权重校准。
