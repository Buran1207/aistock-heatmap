# 二级交易技术分析方法

## 技术指标

本版新增 `scripts/build_technical_signals.py`，从日行情计算：

- MA5 / MA10 / MA20 / MA60 / MA120
- EMA12 / EMA26 / MACD
- RSI14
- BOLL 上轨 / 中轨 / 下轨
- ATR14
- 20D / 60D 收益
- 20D 成交额放大倍数
- 上市以来高点回撤

## 技术状态

### 趋势延续

量化定义：

```text
收盘价 > MA20 > MA60
且 60日收益 ≥ 15%
```

### 趋势转强

```text
收盘价 > MA20
且 MA10 ≥ MA20
```

### 技术破位

```text
收盘价 < MA20
且 成交额 ≥ 20日均值 1.5倍
```

### 放量过热 / 止盈观察

```text
RSI14 ≥ 75
且 成交额 ≥ 20日均值 1.5倍
且 股价位于近60日价格区间高位
```

### 高位回撤预警

```text
从上市以来高点回撤 ≥ 30%
且 当前价低于 MA60
```

## 买卖触发条件

系统不把技术指标直接等同于买入，而是输出：

- 技术状态
- 技术评分
- 买入触发条件
- 卖出触发条件

例如：

```text
趋势延续：回踩20日线缩量企稳，或放量突破20日高点。
技术破位：暂不买入，等待重新站回20日线。
放量过热：不追高，等待回踩；放量滞涨或跌回5日线下方优先止盈。
```

## 流动性过滤

若 20日平均成交额低于 HK$500万，技术分会明显降级；低流动性股票即使形态好，也只作为观察。
