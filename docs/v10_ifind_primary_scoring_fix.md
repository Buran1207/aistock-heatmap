# v10 iFind主数据链路与未上市评分修复说明

## 1. 未上市/A1评分修复

本版将 A1 分数从“项目质量分”升级为“项目参与预判分”，回答：是否值得在招股前提前建档、研究、准备额度或锚定机会。

关键变化：

- 缺资料不再作为主要扣分项。
- 缺资料只降低评分置信度，并进入“需补数据”列。
- 真正扣分必须来自可观察弱项：行业承接弱、同类 IPO 表现弱、中介历史结果弱、未来交易可做性不足等。
- B 档拆分为 B+重点研究 和 B验证型研究，避免大批公司落入粗糙的“B研究池”。
- 新增相对分位、评分置信度、分数来源、关键扣分项、需补数据列。

## 2. A1评分透明化

一级市场 / IPO 项目决策页面中，现在可以直接看到：

- A1质量/参与预判分
- A1相对分位
- 评分置信度
- 分数来源：行业、公司、同类、中介、可做性、市场窗口
- 关键扣分项
- 需补数据
- 人工研究评分和备注

## 3. iFind行情链路修复

Streamlit 页面现在优先读取：

1. deploy_data/ifind_daily_quotes_raw.csv
2. data/raw_ifind/daily_quotes.csv
3. deploy_data/ipo_daily_quotes_180d.csv（仅兜底）
4. deploy_data/ifind_close_snapshot_raw.csv（当日快照临时补充）

二级页面不应再优先显示旧 Yahoo/Stooq 结果。

## 4. 日期显示修复

所有申请日、上市日、解禁日、行情日等日期字段统一显示到 YYYY-MM-DD。1970-01-01 这类解析失败日期会显示为空，避免误导。
