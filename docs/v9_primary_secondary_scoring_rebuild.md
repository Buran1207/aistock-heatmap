# v9 一级/二级决策重构与精细评分版

## 核心变化

1. **A1 和招股期合并为「一级市场 / IPO项目决策」工作区**
   - 保留两套评分：A1项目参与预判分、招股期参与决策分。
   - A1评分用于提前建档、研究和准备额度；招股期评分用于实际打新/锚定/基石/回避。
   - 已上市公司的历史A1/招股评分放在该工作区的「历史IPO评分回看」，仅用于复盘，不进入当前二级评分。

2. **已上市公司只看「已上市 / 二级交易决策」**
   - 当前二级评分不展示历史IPO评分，避免混淆。
   - 二级评分拆分为 0-30D、31-180D、180D+ 三套规则。
   - 页面内可调整不同阶段的评分权重，调整仅影响当前页面排序，不改原始数据。

3. **技术指标进一步专业化**
   - 新增/强化 KDJ、BOLL、OBV、MFI、ATR。
   - 技术指标不直接作为孤立数字展示，而是转化为交易状态、买入触发、卖出触发和风险扣分。

4. **解禁不再单独评分**
   - 解禁作为二级交易评分中的风险扣分和评级上限约束。
   - 正式评分只使用 iFind 精确解禁事件；上市日 + 6/12个月估算不进入正式评分。

5. **数据源与解析修复**
   - 修复 iFinDPy.THSData 返回对象解析，减少 THS_DR 静态表 failed_keep_old。
   - 技术信号优先使用 iFind 日行情与 iFind 快照；Yahoo/Stooq 旧行情只作为兜底。
   - iFind 快照可临时补充当日行情，次日由正式日K覆盖。

6. **显示口径优化**
   - 所有申请日、上市日、解禁日等日期字段显示到日，不显示小时分钟。
   - 人工研究打分嵌入一级市场工作区，不再单独成页。
   - 增加评分置信度和“为什么不是更高评级”。
