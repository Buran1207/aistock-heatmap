# 评分方法说明

## A1 / 未上市项目评分

A1 项目质量分用于判断一家公司未来 IPO 是否值得提前投入研究资源、准备额度和跟踪发行窗口。它不是当前买入建议，也不由申请状态主导。

### 默认权重

| 维度 | 默认权重 | 说明 |
|---|---:|---|
| 行业与港股市场偏好 | 35% | 港股资金是否愿意买这个行业、同类 IPO 是否有赚钱效应和流动性 |
| 公司稀缺性与基本面潜力 | 20% | 是否具备稀缺性、龙头属性、成长性和清晰商业模式 |
| 保荐人 / 中介质量 | 15% | 保荐人、整体协调人、账簿管理人历史项目表现 |
| 历史同类 IPO 表现 | 15% | 同行业、同板块、同类型项目的首日、20 日、60 日表现 |
| 未来交易可做性 | 10% | 预计市场关注度、流动性、二级交易空间 |
| 当前市场窗口 | 5% | 近期港股 IPO 市场赚钱效应和风险偏好 |

### 不纳入 A1 项目质量分的因素

- 是否刚失效
- 是否多次递表
- 是否处理中
- 是否已通过聆讯

这些属于项目管理信息，不属于项目质量本身。系统会在 A1 页面显示最新状态、递表次数、状态提示和下一动作，但不把它们放进项目质量分。

### A1 输出含义

| 质量分 | 研究优先级 | 未来 IPO 参与倾向 |
|---:|---|---|
| ≥80 | A 重点跟踪 | 拟重点参与，等待发行验证 |
| 70-79 | B+ 重点研究 | 倾向参与，需看定价 |
| 60-69 | B 研究池 | 可参与，需强发行信号 |
| 50-59 | C 观察 | 暂不主动参与 |
| <50 | D 低优先级 | 暂不参与 |

## 当前投资状态 C 分类

| 分类 | 含义 |
|---|---|
| C1 等待发行资料 | A1 / 聆讯后，尚未披露完整发行结构 |
| C2 等待配发/暗盘 | 已有发行资料，但等待配发结果、孖展/中签、暗盘确认 |
| C3 等待上市确认 | 即将上市，等待首日价格和成交确认 |
| C4 等待二级买点 | 已上市但不适合追高，等待深 V、回踩或趋势确认 |
| C5 等待风险释放 | 解禁、破发、流动性或数据风险未解除 |
| C6 失效观察 | 未上市但最新申请失效，等待重新递表或项目进展 |

## 权重自定义

“评分标准与权重设置”页面提供 A1 评分滑块。调整后，系统会重新计算 A1 未上市项目排序。该设置用于研究和汇报场景，不会改写原始数据文件。
