# 给领导看的评分口径

系统不是一个分数打天下，而是三阶段评分：

1. A1 项目质量分：判断未上市项目未来 IPO 是否值得提前研究。
2. 招股期参与分：判断一级市场是否打新、锚定、基石、小额参与、只看二级或回避。
3. 二级交易评分：判断已上市公司现在能否买、等、卖、止盈或回避。

## 科学性原则

- 阶段隔离：A1 不用未来发行和上市后数据；招股期不用上市后行情；二级交易才用技术指标。
- 量化定义：深 V、放量突破、放量滞涨、破发弱势、高位回撤、解禁压力倍数均有阈值。
- 权重透明：每个阶段页面展示权重和因子。
- 可验证：回测高分组是否跑赢低分组、破发率是否更低、最大回撤是否更小。
- 可校准：无效因子降权，有效因子升权。

## 解禁口径

解禁不单独评分，而是二级交易风险扣分。正式评分只使用 iFind 精确解禁明细。上市日+6/12个月估算只作为复核提示。

## 行情口径

主行情源为 iFind 日行情和收盘快照。Yahoo/Stooq 只作为兜底，若仅有免费源或行情未更新，则降低置信度或不输出正式交易建议。
